# Google Analytics 4 (GA4) 설정 가이드

## 1. Google Analytics 계정 생성

1. [Google Analytics](https://analytics.google.com/) 접속
2. **측정 시작** 클릭
3. 계정 이름 입력 (예: "제로냥")
4. 속성 이름 입력 (예: "zero-nyang")
5. 시간대 선택: **대한민국 (GMT+09:00)**
6. 통화 선택: **대한민국 원 (₩)**
7. 비즈니스 정보 입력
8. **만들기** 클릭

## 2. 측정 ID 확인

1. **데이터 스트림** 선택
2. **웹** 선택
3. 웹사이트 URL 입력: `https://zero-nyang.vercel.app`
4. 스트림 이름 입력: "제로냥 웹사이트"
5. **스트림 만들기** 클릭
6. **측정 ID** 복사 (예: `G-XXXXXXXXXX`)

## 3. 코드에 통합

### 방법 A: index.html에 직접 추가 (간단)

`index.html`의 `<head>`에 추가합니다.

### 방법 B: React 컴포넌트로 추가 (권장)

더 세밀한 제어가 가능합니다.

## 4. 이벤트 추적 설정

좋아요 클릭, 이메일 제출 등의 이벤트를 추적할 수 있습니다.

## 5. 대시보드 확인

Google Analytics 대시보드에서:
- 실시간 사용자
- 페이지뷰
- 사용자 행동 흐름
- 전환 이벤트
- 지리적 위치
- 디바이스 정보
등을 확인할 수 있습니다.
